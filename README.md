# Corporate-Data-Analyzer
- Corporate Data Analyzer is a simple Python-based desktop application that helps users analyze data from Excel or CSV files. It allows users to create summarized reports using GroupBy and aggregation functions, preview results in a table, visualize data with different chart types and export both reports and charts.

---

# ✅ Project Features

📂 Load CSV and Excel (.csv/.xlsx) files

📊 Automatically detect text and numeric columns

🔍 Group data by a selected column

➕ Apply aggregations: Sum, Mean, Max, Min, Count, Median

📋 Preview summarized report in a table

📈 Create charts: Bar, Column, Line, and Pie

🖼️ Export charts as PNG image

📁 Export reports as Excel (.xlsx) or CSV (.csv)

🧹 Handles numeric values with commas (e.g., 12,000 → 12000)

🖥️ User-friendly Tkinter GUI
